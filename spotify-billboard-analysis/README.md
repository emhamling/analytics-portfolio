# Billboard and Spotify Music Analytics Project

This project analyzed what characteristics are associated with Billboard Hot 100 success by combining Billboard chart data with Spotify audio feature data.

The project involved web scraping, data cleaning, dataset integration, hypothesis testing, machine learning, and data visualization.

Tools used:
- Python
- Pandas
- Selenium
- Machine learning models
- Data visualization

Key contributions:
- Scraped Billboard Hot 100 data
- Merged chart data with Spotify song features
- Explored relationships between song traits and chart performance
- Built models and visualizations to explain what makes a hit song

Skills demonstrated:
- Web scraping
- Data wrangling
- Feature engineering
- Predictive modeling
- Visualization storytelling
