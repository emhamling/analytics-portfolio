# Winnebago Connect Dashboard Project

This project focused on turning connected RV telemetry data into clear business insights using Power BI.

I helped build an interactive dashboard that analyzed camping behavior, boondocking activity, travel usage, power consumption, alerts, and feature engagement across Winnebago Connect data.

Tools used:
- Power BI
- DAX
- Excel
- Data modeling

Key contributions:
- Built and tested DAX measures to classify camping and boondocking activity
- Integrated telemetry, alert, geo, and command datasets using VIN and Date keys
- Created dashboard pages for camping, travel, power, alerts, and commands
- Helped translate technical data into business insights for stakeholders

Skills demonstrated:
- Dashboard development
- Data modeling
- Business analytics
- Data storytelling
