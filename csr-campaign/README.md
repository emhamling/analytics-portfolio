# Corporate Social Responsibility PR Campaign Proposal

This project developed a strategic communications campaign centered on a partnership between the Iowa Wolves and Up2Us Sports.

The campaign was designed to promote youth mentorship through sports while increasing community impact and strengthening organizational social responsibility.

Tools used:
- Strategic communications planning
- Campaign development
- Research and messaging design

Key contributions:
- Created campaign goals, target publics, and key messages
- Built an execution timeline and promotional calendar
- Developed community engagement tactics and evaluation metrics
- Designed communication deliverables including social media content and an infographic

Skills demonstrated:
- Strategic planning
- Public relations
- Community engagement
- Message development
