# Trine University Athletics Facilities Analysis

This project evaluated Trine University’s athletic and recreation facilities, recent improvements, and future development opportunities.

The analysis focused on varsity athletics, recreational usage, student involvement, and facility planning needs within an NCAA Division III environment.

Tools used:
- Facility analysis
- Strategic evaluation
- Research writing

Key contributions:
- Reviewed existing athletic and recreation facilities
- Identified areas for growth, maintenance, and accessibility improvement
- Evaluated participation trends and student usage
- Developed recommendations for future facility investment

Skills demonstrated:
- Sports operations analysis
- Facility planning
- Strategic evaluation
- Research communication
