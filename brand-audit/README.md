# Brand Audit and Analysis Project

This project focused on evaluating a sports brand’s communication strategy, audience engagement, and positioning.

The goal was to identify brand strengths, areas for improvement, and opportunities for more effective fan and community engagement.

Tools used:
- Brand analysis
- Market research
- Audience evaluation
- Strategic recommendations

Key contributions:
- Assessed brand communication assets and audience alignment
- Evaluated strengths and weaknesses in the current strategy
- Provided recommendations for stronger engagement and positioning

Skills demonstrated:
- Brand strategy
- Market research
- Audience analysis
- Strategic communication
